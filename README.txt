Для владельцев убунты:
Проверяем наличие curl-а.

curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt-get install nodejs


Для всех:
После установки, чекаем версии (в cmd/терминале):
node -v
npm -v

Далее заходим в текущую папку, где находится этот файл, и через консоль делаем npm install. Ждем, пока все не установит.

Далее можем работать в проекте. Все, что мы можем редактировать находится в папке src, все, куда будут складываться файлы - в папку dist. Изменять что-то в папке dist бесполезно т.к. она перезаписываема (кроме файла index.html).

Команда npm run dev нам запустит сервер для разработки, и отдачи файлов в режиме просмотра изменений этих самых файлов, и откроет корневую страницу. Она настроена на /dist/index.html (т.е. это значит, что сервер отдаст страницу index.html)

Когда мы вносим любые изменения в любой файл в /src, сервер автоматом подтащит эти изменения и отобразит в браузере, самостоятельно перезагрузив страницу.
Развлекайтесь.
